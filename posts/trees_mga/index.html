A cidade

Maringá é uma cidade paranaense com 409 mil habitantes segundo o Censo 2022 conhecida nacionalmente pela flora abundante, onde 97,5% das edificações da cidade tinha uma ávore próxima (pesquisar número de árvores ou áreas verdes próximas de residências no censo 2010!!!).

A cidade foi planejada em 1947 como Cidade-Jardim, onde os parques e a vegetação permeiam a cidade idealizando um cinturão verde. O planejamento também pode ser visto com as vias atuais que seguem um padrão Norte-Sul e Leste-Oeste bem evidente.

Mas as as vias e parques não contam toda a história…

As árvores

Com acesso aos dados georreferenciados de todas as árvores de Maringá (disclaimer sobre os dados!!, são 140 mil), cada árvore pode ser visualizada como um ponto no mapa!

Neste mapa não existe nada além de pontos para cada árvore. O conjunto delas forma exatamente o desenho das vias da cidade.

A partir dos microdados de cada árvore na cidade, percebe-se a dominância de Sibipirunas na cidade (23,6%), seguida de Oitis (15,3%). Das 142 mil árvores catalogadas, cerca de 17 mil não puderam ser identificadas.

cont_setores <- trees_dataset2 %>% 
  filter(!is.na(setor),
         setor %in% c("Central",
                      "Nordeste",
                      "Noroeste",
                      "Sudeste",
                      "Sudoeste")) %>% 
  group_by(setor) %>% 
  count()

df_sf <- sf::st_as_sf(trees_dataset2, coords = c('longitude', 'latitude'))

centroides <-  df_sf %>% 
  filter(!is.na(setor),
         setor %in% c("Central",
                      "Nordeste",
                      "Noroeste",
                      "Sudeste",
                      "Sudoeste")) %>% 
  group_by(setor) %>% 
  summarize(geometry = sf::st_union(geometry)) %>% 
  sf::st_centroid() %>% 
  sf::st_set_crs("WGS84") %>% 
  left_join(cont_setores)


setores_map <- trees_dataset2 %>% 
  filter(!is.na(setor),
         setor %in% c("Central",
                      "Nordeste",
                      "Noroeste",
                      "Sudeste",
                      "Sudoeste")) %>% 
  ggplot(aes(x=longitude, y=latitude, color = setor))+
  geom_point(size = 0.1)+
  geom_sf_label(data = centroides,
               aes(label = paste0(setor,"\n",n," árvores"))),
               inherit.aes = FALSE,
               family = "Open Sans")+
  scale_color_brewer(type = "qual", 
                     palette = "Set1")+
  theme_void()+
  theme(
    plot.background = element_rect(fill = "#ffffff", color = NA),
    legend.position = "none"
  )+
  coord_sf(xlim= c(-52,-51.87), 
           ylim=c(-23.48,-23.36),
           expand = FALSE,
           crs = "WGS84")

ggsave(plot = setores_map,
       "figures/setores_map.png",
       width = 6, height = 6, dpi = 600)
